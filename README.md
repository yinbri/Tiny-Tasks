# Tiny Tasks API

A minimal REST API built with FastAPI and SQLite that supports managing users and their tasks.  
This project demonstrates clean REST design, database modeling, request validation, and proper error handling.

## Features
- Create and list users
- Create and list tasks
- List tasks for a specific user
- Delete users
- Automatic request validation and interactive API docs

## Tech Stack
- Python
- FastAPI
- SQLAlchemy
- SQLite
